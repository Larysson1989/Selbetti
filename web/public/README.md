# Logos

Coloque aqui os arquivos reais das logos, com **exatamente** esses nomes
(o código em `app/login/page.tsx` já referencia esses caminhos):

- `logo-pequeno-principe.png` — logo do Hospital Pequeno Príncipe
- `logo-selbetti.png` — logo da Selbetti

Recomendado: PNG com fundo transparente, altura mínima de ~200px (o CSS
redimensiona automaticamente, mas parte de uma imagem maior fica mais nítido).

Depois de colocar os arquivos aqui, pode apagar este README.md (ele é só
uma instrução, não faz parte do site).
